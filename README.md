# so_long 42 Madrid
The objective of the project is to learn the basics of a 2D game.

## Installation
- Clone the repo
```bash
git clone https://github.com/danielfdez17/so_long.git
cd so_long
```
## Build
If you have make installed in your system, just build the project with make. Otherwise, you will need to install it.
```bash
make
```

## Usage
It is necessary to indicate a file name.
```bash
./so_long files/test.ber
```
